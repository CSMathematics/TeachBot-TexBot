# /hints - Υποδείξεις

$ARGUMENTS

---

## Purpose

Δημιουργία κλιμακωτών υποδείξεων (3 επιπέδων) για ασκήσεις, χρησιμοποιώντας τον `hint-generator` agent.

---

## Output

```
Άσκηση 1:
💡 Ξεκινήστε υπολογίζοντας την παράγωγο.
💡 Θυμηθείτε: (f·g)' = f'·g + f·g'

Άσκηση 2:
💡 Βρείτε πρώτα το πεδίο ορισμού.
```

---

## Examples

```
/hints worksheet_paragogos.tex
/hints [paste exercise]
```
