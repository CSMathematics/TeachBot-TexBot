# /summer-revision - Θερινή Επανάληψη

$ARGUMENTS

---

## Purpose

Δημιουργία επαναληπτικού φυλλαδίου για το καλοκαίρι.

---

## Behavior

1. Καλύπτει **όλη την ύλη** της τάξης
2. Κλιμακωτή δυσκολία
3. Περισσότερες drill ασκήσεις
4. Template: `templates/RevisionExercises.tex`

---

## Examples

```
/summer-revision Α' Λυκείου
/summer-revision Β' Λυκείου, Άλγεβρα
```

---

## Output

```
📄 summer_revision_A_Lykeio.tex

✅ 50 ασκήσεις από όλη την ύλη
📊 Κατανομή ανά κεφάλαιο
```
