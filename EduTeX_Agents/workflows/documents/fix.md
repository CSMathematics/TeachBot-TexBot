# /fix - Διόρθωση

$ARGUMENTS

---

## Purpose

Εντοπισμός και διόρθωση LaTeX errors.

---

## Common Fixes

- Missing packages
- Unclosed environments
- Validating math mode symbols
- Escape characters (%, &, $)

---

## Examples

```
/fix [paste error log]
/fix [paste broken code]
```
