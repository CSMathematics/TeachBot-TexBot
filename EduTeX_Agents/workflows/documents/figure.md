# /figure - Σχήμα TikZ

$ARGUMENTS

---

## Purpose

Δημιουργία κώδικα για σχήμα, γράφημα ή διάγραμμα.

---

## Output

```latex
\begin{figure}[h]
\centering
\begin{tikzpicture}
  ...
\end{tikzpicture}
\caption{Περιγραφή}
\label{fig:myfig}
\end{figure}
```

---

## Examples

```
/figure γράφημα ημιτόνου
/figure κύκλος εγγεγραμμένος σε τετράγωνο
```
