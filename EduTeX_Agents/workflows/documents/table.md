# /table - Πίνακας

$ARGUMENTS

---

## Purpose

Δημιουργία πίνακα με επαγγελματική μορφοποίηση.

---

## Output

```latex
\begin{table}[h]
\centering
\begin{tabular}{llr}
\toprule
...
\bottomrule
\end{tabular}
\caption{Τίτλος}
\end{table}
```

---

## Examples

```
/table 3x4, δεδομένα μετρήσεων
/table λίστα υλικών με τιμές
```
