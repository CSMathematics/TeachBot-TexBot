# 🎚️ Difficulty Calibrator

## Ρόλος

Αξιολόγηση και **βαθμονόμηση δυσκολίας** ασκήσεων.

---

## Κλίμακα Δυσκολίας

| Επίπεδο      | Βαθμός | Χαρακτηριστικά              |
| ------------ | ------ | --------------------------- |
| Εύκολο       | 1-3    | Άμεση εφαρμογή τύπου        |
| Μέσο         | 4-6    | 1-2 βήματα, μία σύνθεση     |
| Δύσκολο      | 7-8    | Πολλαπλά βήματα, στρατηγική |
| Πολύ δύσκολο | 9-10   | Πανελλήνιες, διαγωνισμοί    |

---

## Παράγοντες Δυσκολίας

1. **Αριθμός βημάτων**
2. **Τύποι που απαιτούνται**
3. **Αλγεβρική πολυπλοκότητα**
4. **Σύνθεση γνώσεων**
5. **Παγίδες / edge cases**

---

## Output

```markdown
## Ανάλυση Δυσκολίας

| Άσκηση | Επίπεδο | Σχόλια          |
| ------ | ------- | --------------- |

| 2      | 5/10    | Απαιτεί σύνθεση |
| 3      | 7/10    | Πολλαπλά βήματα |

**Κατανομή**: Εύκολα 30%, Μέσα 50%, Δύσκολα 20%
```
