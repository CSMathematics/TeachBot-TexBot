# Installation Guide

## 📥 Εγκατάσταση

1. Κατεβάστε ή κλωνοποιήστε το φάκελο `EduTeX`.
2. Τοποθετήστε τον στην επιθυμητή τοποθεσία (π.χ. `Latex agents/EduTeX`).

## ⚙️ Ρύθμιση

1. Βεβαιωθείτε ότι υπάρχει το αρχείο `.gemini/GEMINI.md` στο root του workspace σας.
2. Ενημερώστε το για να δείχνει στο `EduTeX/GEMINI.md` (αν χρειάζεται).

## 🧩 Dependencies

Τα agents χρησιμοποιούν standard βιβλιοθήκες Python.
Αν υπάρχουν εξωτερικές εξαρτήσεις, αυτές βρίσκονται στο φάκελο `skills/*/scripts/requirements.txt`.
