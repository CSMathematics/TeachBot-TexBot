# EduTeX 🎓

> Unified AI LaTeX Assistant for Education & Documents

Ένα ενιαίο σύστημα AI πρακτόρων (agents) που συνδυάζει:

1. **Education Domain**: Δημιουργία εκπαιδευτικού υλικού (πρώην TeachBot)
2. **Documents Domain**: Δημιουργία επιστημονικών εγγράφων (πρώην TexBot)

---

## 🚀 Quick Start

1. Ανοίξτε το φάκελο `EduTeX` στο AI environment σας.
2. Το αρχείο `GEMINI.md` είναι το entry point.
3. Χρησιμοποιήστε slash commands όπως `/worksheet` ή `/document`.

---

## ✨ Features

### Education
- Δημιουργία φυλλαδίων ασκήσεων
- Λύσεις βήμα-βήμα
- Διαγωνίσματα και θέματα Πανελληνίων
- Έλεγχος ύλης και παιδαγωγικών κανόνων

### Documents
- Συγγραφή άρθρων και αναφορών
- Δημιουργία σχημάτων TikZ
- Μορφοποίηση πινάκων
- Παρουσιάσεις Beamer
