# 🧑‍🏫 TeachBot Documentation

> Ο προσωπικός σου βοηθός για δημιουργία εκπαιδευτικού υλικού Μαθηματικών.

---

## 🚀 Βασικές Εντολές

| Εντολή         | Τι κάνει                              | Παράδειγμα                                           |
| -------------- | ------------------------------------- | ---------------------------------------------------- |
| `/worksheet`   | Φτιάχνει **φυλλάδιο ασκήσεων**        | `/worksheet Παράγωγος, 15 ασκήσεις, μέτρια δυσκολία` |
| `/exam`        | Φτιάχνει **διαγώνισμα** (120' ή 180') | `/exam Όρια, 120/180 λεπτά, Β' Λυκείου`              |
| `/solutions`   | Φτιάχνει **λύσεις** για αρχείο        | `/solutions worksheet.tex`                           |
| `/panhellenic` | Φτιάχνει θέματα **Πανελληνίων**       | `/panhellenic Διαφορικός Λογισμός`                   |

## 🛠️ Εργαλεία Καθηγητή

| Εντολή       | Τι κάνει                                | Παράδειγμα                            |
| ------------ | --------------------------------------- | ------------------------------------- |
| `/calibrate` | Ελέγχει τη **δυσκολία** ασκήσεων        | `/calibrate exam.tex`                 |
| `/rubric`    | Φτιάχνει **οδηγό βαθμολόγησης**         | `/rubric exam.tex`                    |
| `/variant`   | Φτιάχνει **παραλλαγές** άσκησης         | `/variant [paste code], 3 παραλλαγές` |
| `/split`     | Χωρίζει σε **Teacher/Student versions** | `/split worksheet.tex`                |
| `/review`    | Ελέγχει αν είναι **εντός ύλης**         | `/review exam.tex, Γ' Λυκείου`        |
