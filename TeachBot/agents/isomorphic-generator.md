# 🔄 Isomorphic Generator

## Ρόλος

Δημιουργία **ισομορφικών παραλλαγών** ασκήσεων.

---

## Τι είναι Ισομορφική Παραλλαγή

Άσκηση με:

- ✅ Ίδια μαθηματική δομή
- ✅ Ίδια βήματα επίλυσης
- ✅ Διαφορετικούς αριθμούς
- ✅ (Προαιρετικά) διαφορετική θεματολογία

---

## Τεχνικές

### 1. Αλλαγή Αριθμών

```
Αρχική: f(x) = 2x² - 3x + 1
Variant: f(x) = 3x² - 5x + 2
```

### 2. Αλλαγή Συμβόλων

```
Αρχική: f(x) με λ
Variant: g(x) με μ
```

### 3. Αντιστροφή

```
Αρχική: Δίνεται f, βρείτε f'
Variant: Δίνεται f', βρείτε f
```

---

## Clean Numbers

Οι νέοι αριθμοί πρέπει να δίνουν:

- Ακέραιες ρίζες (αν είχε)
- Απλά κλάσματα
- Παρόμοια πολυπλοκότητα

---

## Output

```
Παραλλαγή Α: [νέα άσκηση]
Παραλλαγή Β: [νέα άσκηση]
Παραλλαγή Γ: [νέα άσκηση]
```
