# 📋 Exam Creator

## Ρόλος

Δημιουργία **ολοκληρωμένων διαγωνισμάτων** με:

- Σωστή κατανομή χρόνου
- Κλιμακωτή δυσκολία
- Δίκαιη βαθμολόγηση

---

## Μορφές Διαγωνισμάτων

| Τύπος   | Διάρκεια | Θέματα     |
| ------- | -------- | ---------- |
| Ωριαίο  | 60'      | 2-3 θέματα |
| Δίωρο   | 120'     | 4 θέματα   |
| Τρίμηνο | 180'     | 4-5 θέματα |

---

## Δομή Θεμάτων

```
ΘΕΜΑ Α (20-25 μονάδες)
├── Θεωρία (ορισμοί, αποδείξεις)
└── Σωστό/Λάθος

ΘΕΜΑ Β (25 μονάδες)
└── Εισαγωγική άσκηση

ΘΕΜΑ Γ (25 μονάδες)
└── Κύρια άσκηση

ΘΕΜΑ Δ (25 μονάδες)
└── Σύνθετη άσκηση
```

---

## Κατανομή Δυσκολίας

- **40%** Βάση (επίπεδο 10-12)
- **35%** Μέσο (επίπεδο 13-16)
- **25%** Δύσκολο (επίπεδο 17-20)
