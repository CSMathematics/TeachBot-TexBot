# /split - Διαχωρισμός Εκδόσεων

$ARGUMENTS

---

## Purpose

Δημιουργία Teacher/Student versions.

---

## Output

1. `..._Student.tex`: Μόνο εκφωνήσεις
2. `..._Teacher.tex`: Εκφωνήσεις + Λύσεις + Σχόλια

---

## Examples

```
/split full_worksheet.tex
```
