# /prerequisites - Προαπαιτούμενα

$ARGUMENTS

---

## Purpose

Ανάλυση απαραίτητων γνώσεων για ένα κεφάλαιο και δημιουργία refresher box με σύντομες ασκήσεις, χρησιμοποιώντας τον `prerequisite-checker` agent.

---

## Output

LaTeX block με τίτλο "Πριν ξεκινήσετε..." και 2-3 Warm-up ασκήσεις.

---

## Examples

```
/prerequisites Παράγωγοι
/prerequisites Ολοκληρώματα, Γ' Λυκείου
```
