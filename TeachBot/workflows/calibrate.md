# /calibrate - Βαθμονόμηση Δυσκολίας

$ARGUMENTS

---

## Purpose

Αξιολόγηση δυσκολίας ασκήσεων.

---

## Output

```markdown
## Report Δυσκολίας

| Άσκηση | Βαθμός (1-10) | Κατηγορία | Σχόλια  |
| ------ | ------------- | --------- | ------- |
| 1      | 3             | Εύκολο    | Άμεση   |
| 2      | 6             | Μέσο      | Σύνθεση |

**Μέσος Όρος:** 4.5/10 (Κατάλληλο για Β' Λυκείου)
```

---

## Examples

```
/calibrate worksheet.tex
/calibrate [paste exercise]
```
