# /exam - Διαγώνισμα

$ARGUMENTS

---

## Purpose

Δημιουργία ολοκληρωμένου διαγωνίσματος.

---

## Behavior

1. **Parse request**
   - Ύλη/Κεφάλαια
   - Διάρκεια (120' ή 180')
   - Τάξη

2. **Load agents**
   - `exam-creator`
   - `difficulty-calibrator`

3. **Execute**
   - Δημιουργία θεμάτων Α-Δ
   - Κατανομή μονάδων
   - Output: `.tex` αρχείο

---

## Examples

```
/exam Όρια και Συνέχεια, 45', Β' Λυκείου
/exam Παράγωγος, 90 λεπτά
/exam Κεφάλαιο 1-3, τρίμηνο
```

---

## Output

```
📄 exam_oria_45min.tex

✅ Θέμα Α: Θεωρία + Σ/Λ (25 μ.)
✅ Θέμα Β: Άσκηση (25 μ.)
✅ Θέμα Γ: Άσκηση (25 μ.)
✅ Θέμα Δ: Σύνθεση (25 μ.)
```
