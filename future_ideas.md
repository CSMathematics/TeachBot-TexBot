## 🤖 1. Το Οικοσύστημα των AI Agents (The "Brain")
Αντί για ένα απλό chatbot, χρησιμοποιούμε εξειδικευμένους "πράκτορες" που συνεργάζονται αρμονικά για το βέλτιστο αποτέλεσμα.

### 🔹 Core Agents (Τα Θεμέλια)

| Agent | Ρόλος | Super Power |
| :--- | :--- | :--- |
| **A: Curriculum Architect** | Ο "Δάσκαλος" | Αντιλαμβάνεται την ύλη και τη δυσκολία (π.χ. Γ' Λυκείου -> Ολοκληρώματα). |
| **B: Math Generator** | Ο "Κατασκευαστής" | **Twin Generator:** Δημιουργεί ισοδύναμες παραλλαγές για αποφυγή αντιγραφής. |
| **C: Solver & Validator** | Ο "Ελεγκτής" | Επιλύει και ελέγχει αν τα αποτελέσματα είναι "καθαρά" (SymPy validation). |
| **D: LaTeX Typesetter** | Ο "Τυπογράφος" | Μετατρέπει τα πάντα σε άψογο κώδικα (TikZ, PGFPlots, align*). |

### 🔸 Advanced & Future Agents (Το Επόμενο Επίπεδο)

| Agent | Ρόλος | Value Add |
| :--- | :--- | :--- |
| **E: Contextizer** | Ο "Συνδέτης" | Μετατρέπει "ξερές" ασκήσεις σε PISA-style προβλήματα πραγματικού κόσμου. |
| **F: Exam Balancer** | Ο "Ζυγιστής" | Ελέγχει την κατανομή δυσκολίας και την κάλυψη της ύλης (Bloom's Taxonomy). |
| **G: Style Mimic** | Ο "Μιμητής" | Προσαρμόζει το ύφος (Πανελλήνιες, Ευκλείδης, Σχολικό). |
| **H: Student Simulator** | Ο "Μαθητής" | Προβλέπει παγίδες και σημεία που θα "κολλήσουν" οι μαθητές. |
| **I: Hint Designer** | Ο "Βοηθός" | Δημιουργεί κλιμακωτές υποδείξεις (ιδέα -> μεθοδολογία -> σχεδόν λύση). |
| **J: Grader** | Ο "Βαθμολογητής" | Παράγει αναλυτικό grading scheme (rubrics) ανά ερώτημα. |

## 🛠️ 2. Killer Features (Το "X-Factor")

> [!TIP]
> **One Exercise → Full Sheet**
> Δίνεις ΜΙΑ άσκηση και το σύστημα χτίζει αυτόματα ένα πλήρες διαγώνισμα γύρω της (Warm-up -> Core -> Twist).

*   **LaTeX-to-Graph:** Αυτόματη παραγωγή TikZ από συνάρτηση $f(x)$.
*   **OCR Import:** Φωτογραφία άσκησης -> LaTeX -> 3 Παραλλαγές.
*   **Error-Driven Variations:** Παραλλαγές που στοχεύουν σε συγκεκριμένα λάθη (π.χ. "ξεχνάει την απόλυτη τιμή").
*   **Difficulty Slider:** Πραγματική κλιμάκωση (Σχολικό -> Πανελληνίων -> Ολυμπιακό) που αλλάζει τη δομή του συλλογισμού.

## 📊 3. Teacher Superpowers (Analytics & Εργαλεία)

*   **Time Calibration:** Ακριβής εκτίμηση χρόνου ανά ερώτημα και συνολική προειδοποίηση ("Αυτό το διαγώνισμα θέλει 2+ ώρες").
*   **Coverage Heatmap:** Χρωματικός χάρτης που δείχνει ποια κεφάλαια υπερ-εξετάζονται.
*   **Student Version Export:**
    *   Premium layouts με χώρους για λύσεις.
    *   Optional hints μέσω QR codes.
*   **Personalized Remedial Sheet:** Αυτόματο φυλλάδιο ενισχυτικής διδασκαλίας με βάση τα λάθη της τάξης.

## 💻 5. Τεχνικό Stack & R&D

> [!NOTE]
> **MVP Stack:** Python (FastAPI) + React + GPT-4o/Claude 3.5 + SymPy + TeX Live.

*   **LLM Self-Play:** Agents που ανταγωνίζονται μεταξύ τους για τη δημιουργία της "ποιοτικότερης" άσκησης.
*   **Adaptive Difficulty:** Το σύστημα μαθαίνει το επίπεδο κάθε σχολείου και προσαρμόζεται.
*   **Voice → Exam:** "Φτιάξε μου ένα διαγώνισμα σαν το περσινό αλλά 20% πιο δύσκολο".
