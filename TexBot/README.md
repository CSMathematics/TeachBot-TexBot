# 🤖 TexBot Documentation

> Ο βοηθός σου για οτιδήποτε έχει σχέση με LaTeX.

---

## 📄 Δημιουργία Εγγράφων

| Εντολή           | Τι κάνει                                  | Παράδειγμα                                  |
| ---------------- | ----------------------------------------- | ------------------------------------------- |
| `/document`      | Φτιάχνει νέο **έγγραφο** (article/report) | `/document article, Διπλωματική Εργασία`    |
| `/presentation`  | Φτιάχνει **παρουσίαση** (Beamer)          | `/presentation 10 slides, Machine Learning` |
| `/cv`            | Φτιάχνει **βιογραφικό**                   | `/cv academic`                              |
| `/letter`        | Φτιάχνει **επιστολή**                     | `/letter συστατική επιστολή`                |
| `/poster`        | Φτιάχνει **επιστημονική αφίσα**           | `/poster A0, Συνέδριο Φυσικής`              |
| `/greek-article` | Φτιάχνει **ελληνικό άρθρο**               | `/greek-article Θέμα Εργασίας`              |

---

## 🎨 Σχήματα & Πίνακες

| Εντολή      | Τι κάνει                          | Παράδειγμα                           |
| ----------- | --------------------------------- | ------------------------------------ |
| `/figure`   | Φτιάχνει **σχήμα/γράφημα** (TikZ) | `/figure γράφημα ημιτόνου`           |
| `/table`    | Φτιάχνει μορφοποιημένο **πίνακα** | `/table 3x4, αποτελέσματα μετρήσεων` |
| `/equation` | Γράφει πολύπλοκη **εξίσωση**      | `/equation σύστημα 3 εξισώσεων`      |
| `/snippet`  | Δίνει έτοιμο **κώδικα**           | `/snippet integral`                  |

---

## 🛠️ Εργαλεία

| Εντολή          | Τι κάνει                          | Παράδειγμα                          |
| --------------- | --------------------------------- | ----------------------------------- |
| `/convert`      | Μετατρέπει **Word/Text σε LaTeX** | `/convert [paste text]`             |
| `/fix`          | Διορθώνει **λάθη** στον κώδικα    | `/fix [paste broken code]`          |
| `/template`     | Βρίσκει έτοιμα **πρότυπα**        | `/template thesis`                  |
| `/bibliography` | Φτιάχνει **βιβλιογραφία**         | `/bibliography Einstein 1905 paper` |
