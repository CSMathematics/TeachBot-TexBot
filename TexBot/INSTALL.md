# 📦 Οδηγίες Εγκατάστασης TexBot

Για να χρησιμοποιήσετε το TexBot σε ένα **νέο project**:

1. **Δημιουργήστε** ένα φάκελο `.agent` στη ρίζα του νέου project.
2. **Αντιγράψτε** τα περιεχόμενα αυτού του φακέλου μέσα στο `.agent`.

## Αναλυτικά

| Από τον φάκελο `TexBot` | Στο νέο project (`.agent/`) |
| ----------------------- | --------------------------- |
| `agents/*`              | `.agent/agents/`            |
| `skills/*`              | `.agent/skills/`            |
| `workflows/*`           | `.agent/workflows/`         |
| `GEMINI.md`             | `.agent/GEMINI.md`          |

## ✅ Επιβεβαίωση

Μετά την αντιγραφή, πληκτρολογήστε `/status` ή `/document` στο νέο project για να δείτε αν οι agents φορτώθηκαν σωστά.
